🌿 SoulLog
Your Personal Space for Reflection & Growth

🔗 Live Demo: https://soul-log-fawn.vercel.app/


✨ Overview

SoulLog is a modern journaling web application designed to help users reflect, organize thoughts, and track personal growth over time.
It focuses on providing a minimal, distraction-free experience while maintaining a clean and responsive interface for seamless daily use.


🚀 Features

📝 Create & Manage Journal Entries
Easily log daily thoughts, experiences, and reflections
📅 Structured Journaling Flow
Organized entries for better readability and consistency
🎨 Clean & Minimal UI
Designed for focus and ease of use
📱 Responsive Design
Works smoothly across desktop and mobile devices
⚡ Fast & Lightweight Performance
Optimized for quick interactions and smooth navigation



🛠️ Tech Stack

Frontend: React 
Styling: CSS (Custom variables and inline styles)
Icons: Lucide React
Database: Firebase Firestore (NoSQL cloud database)
Authentication: Firebase Auth (Email/password and guest login)
Media Storage: Cloudinary (REST API for image hosting)
Image Processing: Native HTML5 Canvas API (Client-side image compression)
Architecture: Serverless / Backend-as-a-Service (BaaS)

🧠 Problem It Solves

In today’s fast-paced environment, people often lack a simple and consistent way to reflect on their thoughts.
SoulLog solves this by:

Providing a structured journaling platform
Encouraging daily reflection habits
Offering a clean, distraction-free writing space


🎯 Key Learnings

Building responsive UI with modern design principles
Structuring frontend components effectively
Improving user experience through simplicity
Managing project deployment and version control
